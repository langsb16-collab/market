GitHub 설정

GitHub 연동이 필요한 이유

PC와 모바일 간 데이터 자동 동기화를 위해 GitHub Repository를 사용합니다.

  • 공지, 배너, 팝업 데이터가 JSON 파일로 저장됩니다
  • GitHub Pages를 통해 자동으로 PC/모바일에 배포됩니다
  • 데이터 변경 이력이 Git으로 자동 관리됩니다

설정 확인 중...

GitHub Token 생성하기 (repo 권한 필요)
현재 웹사이트가 배포된 GitHub Repository 이름

GitHub Token 생성 방법

  1. GitHub 로그인 후 Token 생성 페이지 접속
  2. Note: "EventBET Admin" 입력
  3. Expiration: "No expiration" 선택 (만료 없음)
  4. Select scopes: "repo" 체크 (전체 Repository 접근 권한)
  5. "Generate token" 클릭
  6. 생성된 Token 복사하여 위 입력창에 붙여넣기
