# EventBET - 정렬 버튼 색상 가이드

## 🎨 각 버튼의 색상

### 1. **최근등록순** 🟣 (보라색)
- **활성화**: 보라색 그라데이션 (#9333ea → #7c3aed)
- **비활성**: 연한 보라색 배경 + 보라색 테두리
- **의미**: 최신 등록된 이슈를 강조

### 2. **결과발표일** 🔵 (파란색)
- **활성화**: 파란색 그라데이션 (#3b82f6 → #2563eb)
- **비활성**: 연한 파란색 배경 + 파란색 테두리
- **의미**: 시간/일정 관련 정렬

### 3. **배팅규모** 🟢 (초록색)
- **활성화**: 초록색 그라데이션 (#10b981 → #059669)
- **비활성**: 연한 초록색 배경 + 초록색 테두리
- **의미**: 금액/재정 관련 정렬

### 4. **이용객 숫자** 🟠 (주황색)
- **활성화**: 주황색 그라데이션 (#f59e0b → #d97706)
- **비활성**: 연한 주황색 배경 + 주황색 테두리
- **의미**: 인기/참여도 관련 정렬

## 💡 디자인 특징

### 활성 상태:
- 강렬한 그라데이션 배경
- 흰색 텍스트
- 그림자 효과 (각 색상에 맞춘 그림자)
- 부드러운 전환 애니메이션

### 비활성 상태:
- 연한 색상 배경 (각 버튼 고유 색상의 8% 투명도)
- 색상 테두리 (각 버튼 고유 색상의 20% 투명도)
- 호버 시 약간의 확대 효과 (1.02배)

### 다크 모드:
- 비활성 버튼의 배경 투명도를 15%로 증가
- 테두리 투명도를 30%로 증가
- 더욱 선명한 대비 효과

## 🎯 사용자 경험 개선

1. **시각적 구분**: 각 버튼이 고유한 색상을 가져 쉽게 구분 가능
2. **직관적 인식**: 색상만으로도 어떤 정렬 옵션인지 파악 가능
3. **접근성 향상**: 색맹 사용자도 아이콘과 텍스트로 구분 가능
4. **일관성 유지**: 모든 버튼이 동일한 디자인 패턴 사용

## 📱 반응형 디자인

- 모바일/태블릿에서도 동일한 색상 적용
- 터치 피드백 향상을 위한 호버 효과 유지
- 가로 스크롤 시에도 색상 유지
